# 🎬 CineMatch: Movie Recommender System

CineMatch is a content-based movie recommendation web app built using **Python**, **Streamlit**, and **Scikit-Learn**. It calculates similarity scores using Cosine Similarity on movie features and fetches dynamic posters via **The Movie Database (TMDb) API**.

## 🚀 Features
* Recommends top 5 similar movies based on content similarity.
* Fetches real-time movie posters using TMDb API.
* Clean and responsive user interface built with Streamlit.

## 🛠️ Tech Stack
* **Frontend/UI:** Streamlit
* **Backend Logic:** Python, Pandas, NumPy
* **Machine Learning:** Scikit-Learn (Cosine Similarity)
* **API Integration:** TMDb API
